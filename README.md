# Trick or Treat
## This is a website inspired by the theme "spoOoky". It's an app, whereby you go knocking on doors and get TRICKED or TREATED. 

## Features
* Click on doors to get candies or tricked
* HighScore calculation
* Progress Bar
* A reset button

## Technologies used
* React 
* CSS
* React Icons
